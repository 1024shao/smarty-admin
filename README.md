# 特性

前端工程化演示项目。

- Monorepo风格 (pnpm workspace)
- 独立组件库
- 独立脚手架工具
- SSG风格文档自动化工具
- seed项目模板

# 目录说明
```bash
│   packages
│   ├── seed # admin空模板
│   ├── docs  # 文档自动化
│   ├── ui # 组件库
│   ├── smarty-ui-vite # 组件库 (Vite版本)
│   ├── admin # admin全量
│   ├── cli # 脚手架工具
```
